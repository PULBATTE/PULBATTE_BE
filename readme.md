## 식집사들을 위한 식물 관리 사이트
- 성향에 맞는 식물을 테스트해보고, 테스트한 식물의 성장도를 기록할 수 있어요
- 식물의 특징을 필터링해서 검색하고 다양한 식물의 정보를 찾아볼 수 있어요
- 키우는 식물을 체계적으로 관리하고 식물과 함께하는 일상을 기록해보세요
- 식물에 대한 정보와 이야기를 식집사 이웃들과 공유해보세요!

## 3. 기능
- 식집사 테스트/ 가이드
  - 16가지 유형의 성격 테스트
  - 6가지 의 식물 추천 및 가이드

- 식물 찾아보기
  - 300여종의 식물 검색
  - 5가지 필터 (초보자, 잎, 열매, 꽃, 다육/선인장)
